## Default Permission

webview 캡처·분석 명령(snapshot / record / set_occlusion / analyze_regions) 허용.

#### This default permission set includes the following:

- `allow-snapshot`
- `allow-record`
- `allow-set-occlusion`
- `allow-analyze-regions`
- `allow-analyze-frame-diffs`
